# ログイン　サインアップ画面の作成

- 前回BEで作ったAPIを使って、ログインとサインアップの実装。
- サインアップでアカウントを作り、ログインでtodoへ移行出来るようにしてる。

# アカウント削除画面の追加とcookieの実装

- ログイン後のtodoの中に、アカウント削除ボタンを追加して
  ボタン押すとアカウントを削除できるように実装。
- アカウント削除するとユーザー情報とtodoがすべて削除されるように
- アカウント削除する時にトークンの期限が切れた場合は、
  弾いてログイン画面へ行くように実装。
- BEで作ったcookieをFEに実装。
  ネットワークにcookieが送られているのを検証済み。
- コメントでいつでも振り返れるように、
  コメントを残しています。
